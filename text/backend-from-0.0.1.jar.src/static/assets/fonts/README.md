将可分发的中文字体文件放在此目录，以便生成“真正矢量且不乱码”的 PDF。

当前代码默认尝试加载：

- `assets/fonts/full-font.ttf`（优先，推荐）
- `assets/fonts/simsunb.ttf`（回退，若该字体在你环境下渲染为空则不要用它）

你可以放入任意支持中文的 TTF，并保持上述文件名；或改 `app.component.ts` 里的加载路径与字体名映射。

说明：
- **如果没有字体文件**：生成 PDF 会自动回退到高 DPI 位图页（不乱码，但放大更糊）。
- **如果提供了字体文件**：中文会以矢量文字写入 PDF，放大依然清晰。
